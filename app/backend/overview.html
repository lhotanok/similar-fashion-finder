Similar Fashion Finder is a web-based tool for searching fashion products by images.
It is designed to help users find similar fashion products by uploading an image of a product they like.
The application uses several image hashing algorithms to compare the uploaded image with a database of fashion products.
The user can then view the similar products and purchase them online.

The backend of the application consists of the following components:

  1. Product Database

    A database of fashion products with image URLs and other metadata.
    Each product is represented as a document in a MongoDB collection.
    Each website has its own collection in the database, such as 'zalando-products' and 'zoot-products'.

  2. Image Processing

    Images are extracted from product data, they are downloaded from remote URLs,
    transformed (cropped to square and resized to a smaller size for more efficient processing)
    and uploaded to the H2 database in form of hashes. There are several hashing algorithms used to hash the images:

    • Difference Hash
    • Perceptive Hash
    • Wavelet Hash
    • RotP Hash

  3. Image Search API

    An API that accepts an image URL, searches for similar images and uses those images to extract similar products from the MongoDB database.
    Those products are then returned to the user along with the similarity score (distance and hamming distance) between the queried image and searched images.
